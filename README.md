# Student House Management System

A web-based system designed to manage student boarding houses, track rental details, and handle monthly utility billing.

---

## Core Features

### 1. Account Management (Quản lý tài khoản)
* Add, edit, and delete user accounts.
* Manage and assign user roles and permissions.

### 2. Room Management (Quản lý phòng)
* View the complete list of boarding rooms.
* Track room availability status and rental prices.

### 3. Contract Management (Quản lý hợp đồng)
* Maintain the list of active rental contracts.
* Monitor contract expiration dates and security deposits.

### 4. Utility Management (Quản lý điện nước)
* Record and view monthly electricity and water meter readings.

### 5. Invoice Management (Quản lý hóa đơn)
* View the list of monthly room and utility bills.
* Track bill payment status (Paid / Unpaid).

### 6. Payment History (Lịch sử thanh toán)
* View detailed records of payment dates and times.
* Track the exact amount of money collected from tenants.

### 7. Statistics & Reports (Thống kê)
* Access overview dashboards and business reports.

---

## Tech Stack
* **Language:** Java
* **Web Technology:** Servlet, JSP, JSTL
* **Database:** Microsoft SQL Server / MySQL
* **Server:** Apache Tomcat
* **UI/UX:** HTML5, CSS3, Bootstrap 5, JavaScript
